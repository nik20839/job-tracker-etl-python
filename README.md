# Job Tracker ETL Automation

This repository demonstrates a hands-on practical use case of Python for web scraping, data cleaning, and ETL-style workflows.

The workflow collects job-related data from websites, structures it using Pandas, removes duplicate records, and stores the cleaned data into CSV format for further analysis.

---

## Features

- Web scraping using BeautifulSoup
- Structured data extraction
- Data cleaning using Pandas
- Duplicate handling
- Incremental loading concepts
- CSV dataset generation
- Basic ETL-style workflow

---

## Technologies Used

- Python
- Pandas
- BeautifulSoup
- Requests

---

## Files Included

- `main.py` → Main scraping and data processing workflow
- `jobs.csv` → Sample structured output dataset
